# Server and Client Side Configurations

**Server Side Configurations**             | **Client Side Configurations**
------------------------------------------ | -----------------------------------------
`npm init -y`                              | `npm i graphql`
`npm i graphql`                            | `npm i @apollo/client`
`npm i mongoose`                           |
`npm i apollo-server`                      |
`npm i typescript`                         |
`npm i @types/node`                        |
`npm i ts-node-dev`                        |
`npm i dotenv`                             |

**For running server:**

`ts-node src/server.ts`
